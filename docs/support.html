Before Contacting Support ก่อนติดต่อฝ่ายสนับสนุน

  1. Make sure you have the latest version of JT Planner
  2. ตรวจสอบว่าคุณมี JT Planner เวอร์ชันล่าสุด
  3. Try restarting the app
  4. ลองรีสตาร์ทแอป
  5. Check the FAQ section on our homepage
  6. ตรวจสอบส่วนคำถามที่พบบ่อยในหน้าหลัก

Reporting a Bug รายงานข้อบกพร่อง

When reporting a bug, please include the following information to help us resolve the issue quickly:

เมื่อรายงานข้อบกพร่อง กรุณาแจ้งข้อมูลต่อไปนี้เพื่อช่วยให้เราแก้ไขปัญหาได้รวดเร็ว:

  • Device model (e.g., iPhone 17 Pro, iPad Air)
  • รุ่นอุปกรณ์ (เช่น iPhone 17 Pro, iPad Air)
  • iOS version (found in Settings → General → About)
  • เวอร์ชัน iOS (ดูได้ที่ ตั้งค่า → ทั่วไป → เกี่ยวกับ)
  • App version (found in JT Planner → Settings → About)
  • เวอร์ชันแอป (ดูได้ที่ JT Planner → ตั้งค่า → เกี่ยวกับ)
  • Steps to reproduce the issue — describe what you were doing when the problem occurred
  • ขั้นตอนในการทำซ้ำปัญหา — อธิบายว่าคุณกำลังทำอะไรเมื่อปัญหาเกิดขึ้น
  • Screenshots if possible — visual examples help us understand the issue
  • ภาพหน้าจอ ถ้าเป็นไปได้ — ตัวอย่างภาพช่วยให้เราเข้าใจปัญหา

Send your bug report on Discord: Join our Discord

ส่งรายงานข้อบกพร่องผ่าน Discord: เข้าร่วม Discord ของเรา

Troubleshooting Common Issues การแก้ไขปัญหาที่พบบ่อย

App crashes or freezes แอปค้างหรือหยุดทำงาน

  • Force quit the app and restart it
  • บังคับปิดแอปแล้วเปิดใหม่
  • Restart your device
  • รีสตาร์ทอุปกรณ์
  • Check for app updates in the App Store
  • ตรวจสอบอัปเดตแอปใน App Store
  • If the issue persists, contact support
  • หากปัญหายังคงอยู่ ติดต่อฝ่ายสนับสนุน

Data not syncing ข้อมูลไม่ซิงค์

  • Make sure you're signed in to your account
  • ตรวจสอบว่าคุณลงชื่อเข้าใช้บัญชีแล้ว
  • Check your internet connection
  • ตรวจสอบการเชื่อมต่ออินเทอร์เน็ต
  • Try signing out and signing back in
  • ลองออกจากระบบแล้วลงชื่อเข้าใช้ใหม่
  • Verify that iCloud sync is enabled in Settings
  • ตรวจสอบว่าเปิดใช้งานการซิงค์ iCloud ในตั้งค่าแล้ว

Offline data not available ข้อมูลออฟไลน์ไม่พร้อมใช้งาน

  • Go to Settings → Offline Data
  • ไปที่ ตั้งค่า → ข้อมูลออฟไลน์
  • Download the prefecture data you need
  • ดาวน์โหลดข้อมูลจังหวัดที่คุณต้องการ
  • Make sure you have sufficient storage space
  • ตรวจสอบว่ามีพื้นที่จัดเก็บเพียงพอ
